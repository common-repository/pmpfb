== Installation ==

1. Upload `pmpfb` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to add fields to PMPro checkout =

Simply drag the desired field type into the position you want. Adjust the settings and save.

= Will it alter the functionality of my existing form creating code =

No it will not

= Does this plugin depend on other plugins? =
Yes, you must have Paid Memberships Pro and Paid Memberships Pro Register Helper Addon(both free) activated.
